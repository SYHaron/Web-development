Onderdeel 1. Selectors & Visual ruler

CSS selector = eerste deel van een CSS-regel.
Het is een patron van elementen en andere termen die de browser vertellen welke HTML elementen moeten worden geselecteerd om de CSS-eigenschappen binnen de regel toe te passen.

Het element of de elementen die door de selector worden geselecteerd, worden 'The subject of the selector' genoemd. Bijvoorbeeld: 
  
                        h1 {
                            color: blue;
                            background-color: yellow;
                            }

                        p{
                            color: red;
                                }

Styling die toegepast wordt toegepast in een bepaalde volgorde.

Het kan voorkomen dat je styling toegepast en vervolgens niet het gewenste resultaat ziet.

Cascade en Inheritance zijn belangrijke concepten om te begrijpen hoe styling werkt.

Onderdeel 2. Het box model

Alles in CSS heeft een zogenaamde 'box' eromheen het begrijpen van de boxen is sleutel naar lay-outs met CSS te kunnen maken of om items met andere items uit te lijnen.

Term 'box model' gebruikt wanneer we het hebben over ontwerpt en layout. Elke box in CSS bestaat uit 'margins', 'border', 'padding' en 'content' oftwel de inhoud.

Als de box-sizing: border-box gebruikt = dan werkt alles in de box, dus wordt die niet groter naar buiten toe.

Onderdeel 3.Display & Positioning

CSS position property bepaalt hoe een HTML element in een document wordt gepositioneerd.

Je kan elk HTML element op elke gewenste locatie plaatsen.

De top, right, bottom, left properties bepalen de uiteindelijke locatie van geplaatste elementen.

Postioning values Uitleg CSS postioning
Static
Postioning volgens normal flow document.
Top, right, bottom, left, z-index properties have no effect>>default value.
Relative
Postioning volgens normal flow document, offset relative to itself based on values (top, right, bottom, left).
Offset beinvloedt niet de positie van andere elementen, space gegeven voor een element in page layout blijft hetzelfde als static.
Absolute
Element verwijdert van normal flow document, geen space gemaakt voor het element in pagina layout.
Positie is relatief aan de initial containing block.
Final positie is bepaald door values (top, right, bottom, left).
Fixed
Dit value, fixed, maakt een new stacking context. In geprinte documenten, element is gepositioneerd op dezelfde plek op elke pagina.
Element is verwijderd van normal flow document, geen space gemaakt ervoor in pagina layout.
Positie is relatief aan de intial containing block bepaald door viewport, behalve als als het ancestors (transform, perspective, filter) property set is to something other than none. In dit geval reageerd de ancestor als contaning block.
Eind positie wordt bepaald door values (top, right, bottom, left).
Sticky
Postioning volgens normal flow.
Offset is relatief aan zijn dichtsbijzijnde scrolling ancestor en containing block (nearset block-level ancestor).
Incl. table-related elements, gebaseerd op values (top, right, bottom, left). Offset beinvloed niet de positie van andere elementen.
Deze value maakt een new stacking context.
Sticky element plakt aan de dichtsbijzijnde ancestor en heeft scrolling mechanisme (gemaakt overflow is hidden, scroll, auto, overlay), zelfs als dichtsbijzijnde ancester geen dichtsbijzijnde scrolling ancestor is.
Effectively inhibits any sticky behaviour.
            De default position in je html is static. Het element wordt gepositioneerd volgens de normale stroom van het document. De eigenschappen boven, rechts, onder, links en z-index hebben geen effect. Dit is de standaardwaarde.
                  
            Position properties:
            Relative: Het element is relatief gepositioneerd ten opzichte van zijn normale positie, dus "left: 20px" voegt 20 pixels toe aan de LINKER-positie van het element
                
            Absolute: Het element is gepositioneerd ten opzichte van het eerste gepositioneerde (niet statische) bovenliggende element
                
            Fixed: Het element is gepositioneerd ten opzichte van het browservenster
                
            Sticky: Het element wordt gepositioneerd op basis van de scrollpositie van de gebruiker.

Onderdeel 4. Pseudo elements

Met CSS kan je stijlen definieren op basis van elementen. Maar soms is het niet genoeg om het gewenste resultaat te krijgen. Als je alleen over element- of attribuut- selectors hebt, bijvoorbeeld opmaak eerste letter/regel van inhoud elementen.

Om dit mogelijk te maken zijn Pseudo-elementen geintorduceerd.

Je kan ze zien als een denkbeeldig element die niet in je document voorkomt, maar wel kan 'stijlen'.

Je tal van interessante ontwerpmogelijkheden toepassen zonder de senmatiek van de opmaak negatief te beinvloeden.

De meest gebruikte pseudo-elementen zijn: "::before en ::after".

Pseudo elementen kan je position: absolute, relative t.o.v. parent element, kan je ze zien als twee extra lagen waar je me kunt spelen.

Tal van toepassingen mogelijk, zoals meervoudige borders, achtergronden, allerlei vormen.

Uitleg meerdere achtergrondne en borders (niet beveiligd).
