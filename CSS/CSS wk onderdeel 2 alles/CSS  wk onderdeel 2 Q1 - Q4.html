Q1. Wat zijn de eigenschappen van block boxes en inline boxes?

Block boxes eigenschappen:
Start op nieuwe regel.
Spreidt zich uit op de gehele regel binnen de container, 100% van beschikbare ruimte.
Breedte en Hoogte wordt aangehouden/gerespecteerd.
Padding, Margin, Border duwen andere elementen weg van de block box.
Het besluit van de verandering van 'display type to inline', elements gebruiken dan het block als hun 'outer display' by default.

Inline boxes eigenschappen:
Start niet op een nieuwe regel.
Breedte & Hoogte instelling geldt niet.
Verticale padding, margin, borders gelden wel, maar duwt andere elementen niet weg.
Horizontale padding, margin, borders gelden wel, maar kan andere 'inline boxes' wegduwen van de box.

Bijvoorbeeld: <a></a>, <span></span>, <em></em>, <strong></strong> zijn elementen die 'Inline' verschijnen by default.

Het type box gebruikt voor bepaalde element is beinvloed door 'display property values' (block / inline) en relateerd aan de 'outer value of display'.

Q4. Leg de volgende termen uit: content, padding, margin, border.

Content
De inhoud van het element. <p></p> = de tekst/paragraaf inhoud.

Padding:
De ruimte tussen content en padding border. De ruimte om de content heen binnen specifieke grenzen.
Je hebt padding-top, padding-right, padding-bottom, padding-left.
Padding shortkey = padding: 100px, 50px, 20px, 30px (ook procenten etc.). Dat is top, right, bottom, left.
1x value in padding = (alles hetzelfde) top & bottom & right & left
2x value in padding = top & bottom, right & left
3x value in padding = top, bottom, right & left
Margin:
Ruimte tussen de grens/border edge en elementen buiten het block/box.
De ruimte om elementen heen.
Bijvoorbeeld: 1x box hebt, eerst Paragraaf element hebt, is het de ruimte tussen paragraaf element en box element.
Border:
Uitleg border
De rand om de block/box heen. Ruimte tussen padding & margin.
Border kan je een specifieke style, breedte, kleur geven.
