Q1/Q2. Kan je 14 voorbeelden geven van pseudo-elementen en leg ze uit.

Pseudo-element
::after (2x ::)
Pseudo-class
:after (1x : )
::after
Syntax: a::after { content: "→"; } = pijltje na een link.
Inline bij default. Maakt een pseudo-element, laaste kind van een selected element. Vaak gebruikt voor toevoeging van cosmetisch content aan element with content property.
Enkel te gebruiken bij boxed elementen/specifieke elementen, niet vervaning boxes/element img, br elementen.
::before
Syntax: ::before = toegepast op 1e kind van selected element.
Inline gebruik, voornamelijk cosmetisch.
::backdrop
Syntax: ::backdrop = toegepast als een box met dezelfde maten als viewport.
Komt automatisch onder elk element dat in full-screen mode.
Incl. elementen met FullscreenAPI, dialog elements
Full screen element worden Lifo (last in-first out) geplaatst, speciale laag in viewport, komt als laaste (dus bovenaan) voor viewport dingen.
Neither inherits from of omgekeerd van andere lementen. Geen limitaties voor welke property je het kan gebruiken.
::cue
syntax: ::cue { selector } = matches WebVTT cues, binnne een element. Kan je de style cpations en andere cues in media met VTT tracks stijlen.
Properties toegepast op alle cues, alsof ze één zijn.
De uitzondering is background en longhand properties geldt per cue zelfstandig, ter verkomen dat created boxes, onbedoelde verberging van grote gebieden van media te verwijden.
::cue-region { }
Syntax: ::cue-region | ::cue-region { selector }
Geldt hetzelfde als ::cue enkel pseudo-element.
::first-letter (:first-letter)
Syntax: ::first-letter
Toepast een stijl aan de eerste letter van eerste regel in een block element.
1e letter niet altijd makkelijk te zien, puntuation (Pi, elke stijl open, close, intial quote, final quote, andere punctuatie).
Bepaalde talen, NL = IJS... ijs.
Gebruik van PE-::before, neemt 1e letter eerste stukje content.
::first-line (:first-line)
Syntax: ::first-line (:first-line)
Toepasbaar op 1e regel van een block-element.
Lengte 1e regel is afhankelijk van verschillende factoren.
