Q1/Q2. Kan je 14 voorbeelden geven van pseudo-elementen en leg ze uit.

Pseudo-element
::after (2x ::)
Pseudo-class
:after (1x : )
::after
Syntax: a::after { content: "→"; } = pijltje na een link.
Inline bij default. Maakt een pseudo-element, laaste kind van een selected element. Vaak gebruikt voor toevoeging van cosmetisch content aan element with content property.
Enkel te gebruiken bij boxed elementen/specifieke elementen, niet vervaning boxes/element img, br elementen.
::before
Syntax: ::before = toegepast op 1e kind van selected element.
Inline gebruik, voornamelijk cosmetisch.
::backdrop
Syntax: ::backdrop = toegepast als een box met dezelfde maten als viewport.
Komt automatisch onder elk element dat in full-screen mode.
Incl. elementen met FullscreenAPI, dialog elements
Full screen element worden Lifo (last in-first out) geplaatst, speciale laag in viewport, komt als laaste (dus bovenaan) voor viewport dingen.
Neither inherits from of omgekeerd van andere lementen. Geen limitaties voor welke property je het kan gebruiken.
::cue
syntax: ::cue { selector } = matches WebVTT cues, binnne een element. Kan je de style cpations en andere cues in media met VTT tracks stijlen.
Properties toegepast op alle cues, alsof ze één zijn.
De uitzondering is background en longhand properties geldt per cue zelfstandig, ter verkomen dat created boxes, onbedoelde verberging van grote gebieden van media te verwijden.
::cue-region { }
Syntax: ::cue-region | ::cue-region { selector }
Geldt hetzelfde als ::cue enkel pseudo-element.
::first-letter (:first-letter)
Syntax: ::first-letter
Toepast een stijl aan de eerste letter van eerste regel in een block element.
1e letter niet altijd makkelijk te zien, puntuation (Pi, elke stijl open, close, intial quote, final quote, andere punctuatie).
Bepaalde talen, NL = IJS... ijs.
Gebruik van PE-::before, neemt 1e letter eerste stukje content.
::first-line (:first-line)
Syntax: ::first-line (:first-line)
Toepasbaar op 1e regel van een block-element.
Lengte 1e regel is afhankelijk van verschillende factoren.
::file-selector-button
Syntax: selector::file-selector-button
Staat voor button of input-tag of type="file".
::file-selector-button = een element opzichtzelf, dus je kan stijlen met kleurtjes.
::grammar-error
Syntax: ::grammar-error
PE is een stuk text dat is gemarkeerd als grammatisch incorrect.
Nog NIET werkende in webbrowers.
::marker
Syntax: ::marker
Selecteert de marker box van een list item.
Kan je de 'bullet-points' stijlen van een lijst.
::part()
Syntax: ::part( <ident>+ )
Represents any element binnen een shadow tree dat een matching part attribute.
::placeholder
Syntax: ::placeholder
Staat voor placeholder text in input-tag of textarea-tag element.
Enkel de subset CSS properties dat ::first-line pseudo-element gebruiken, kan je gebruiken in combinatie van ::placeholder pseudo-element in zijn selector.
::selection
Syntax: ::-moz-selection // ::selection
Applies style aan het gedeelte van het document dat is highlighted bij de gebruiker (muis klik-sleep).
::slotted()
Syntax: ::slotted()
Toepasbaar op elk element met locatie in een slot binnen een HTML template.
Dit werkt enkel binnen CSS geplaatst in een Shadow DOM. Het targets enkel elementen niet text node.
                ::slotted( <compound-selector-list> )
                    where 
                        <compound-selector-list> = <compound-selector>#
                        
                    where

                        <subclass-selector> = <id-selector> | <class-selector> | <attribute-selector> | <pseudo-class-selector>
                        <pseudo-element-selector> = ':' <pseudo-class-selector>
                        <pseudo-class-selector> = ':' <ident-token> | ':' <function-token> <any-value> ')'

                        <wq-name> = <ns-prefix>? <ident-token>
                        <ns-prefix> = [ <ident-token> | '*' ]?  | 
                        <id-selector> = <hash-token>
                        <class-selector> = '.' <ident-token>
                        <attribute-selector> = '[' <wq-name> ']' | '[' <wq-name> <attr-matcher> [ <string-token> | <ident-token> ] <attr-modifier>? ']'
                        
                    where 
                        <attr-matcher> = [ '~' |  |  | '^' | '$' | '*' ]? '='
                        <attr-modifier> = i | s
::spelling-error
Syntax: ::spelling-error
Een text gedeelte dat wordt onderstreept als het incorrect spelling is.
Toepasbaar op CSS properties: color, background-color, cursor, caret-color, outline en zijn longhands, text-decoration en zijn bijbehorende propeties, text-emphasis-color, text-shadow.
Nog NIET werkende in webbrowers.

::first-line (:first-line) - pseudo element

Toepasbaar: All font properties, all background properties, color property, text-decoration(-color/-inline/-style). word-spacing, letter-spacing, text-decoration, text-transform, line-height.

selector::file-selector-button- pseudo element

::grammar-error - pseudo element

Voorbeeld: grammatisch error =rood onderstreept.

::marker - pseudo element

Toepasbaar op CSS properties: all font properties, white-space, color, text-combine-upricht, unicode-bidi, direction, content, all animation, all transition.

::part( <ident>+ ) - pseudo element

Voorbeeld met JavaScript: Tab1 Tab2 Tab3 = elk wordt dan zwart+text blauw als je muis-hover doet.
Dus dat je kan zien wat je gaat 'selecteren'. CSS om de tabs naast elkaar te zetten.

:::placeholder - pseudo element

Voorbeelden:
Color contrast = text leesbaar maken, door kleur-verschil/contrast. Gebruik: dat je bepaalde informatie kan blijven hebben, maar toch iets kan typen.
Windows high contrast mode = placeholder text heeft dezelfde style als user-getypte text. Kan in deze modus leesbaarheid beinvloeden.
Labels = Geen vervanging voor label-tag. Zonder label kunnen screenreader programmes geen text lezen.
Style placeholders: text is rood.
Opaque text = opacity tot welk level element achter text is ontzichtbaar.

::selection - pseudo element

Toepasbaar op CSS properties: color, background-color, cursor, caret-color, outline en zijn longhands, text-decoration en bijbehorende properties, text-shadow (background-image is ignored).
Voorbeeld: dat je text wordt highlighted in kleur naar keuze als je met muis klik-sleep doet.
Deze regel tekst is rood/goud als je klikt-sleep, omdat het CSS selection hele document is en niet specifieke class/id naam.
