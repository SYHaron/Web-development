Q1. Je kunt op 3 verschillende manieren CSS in HTML inladen. Welke zijn dat en leg ze uit:

External CSS: je kunt het uiterlijk van een hele website veranderen door slechts één bestand te wijzigen.

Elke HTML-pagina moet een verwijzing bevatten naar het externe stijlbladbestand in het <link> -element, in de head-sectie. Een externe stylesheet kan in elke teksteditor worden geschreven en moet worden opgeslagen met de extensie .css.

Internal CSS: Een interne stijlpagina kan worden gebruikt als een enkele HTML-pagina een unieke stijl heeft. De interne stijl wordt gedefinieerd in het <style> -element, in de head-sectie.

Inline CSS: Een inline stijl kan worden gebruikt om een unieke stijl toe te passen op een enkel element. Om inline stijlen te gebruiken, voegt u het style-attribuut toe aan het relevante element. Het style-attribuut kan elke CSS-eigenschap bevatten.
