1. Wat is het verschil tussen het internet en het world wide web?

World Wide Web, is een stukje van het internet.
Het internet is een wereldwijd netwerk die miljoenen computers met elkaar verbindt, denk aan hardware, routers, kabels, computers enz. Het WWW is een grote verzameling van pagina's van informatie die je met een simpele muisklik kan bekijken.

5. Op HTML elementen kun je attributen gebruiken. Wat zijn HTML attributen? Geef minimaal 5 attributen en maak voorbeelden waar je de attributen in toepast.

Een attribuut geeft extra informatie over een element. Attributen worden altijd geschreven in de start-tag en worden gevolgd door een gelijk teken. 

        - Style attribute 
            <h2 style="background-color: #ff0000;">Mijn eerste HTML pagina.</h2>
        - class attribute
            <p class="important">Let op dit is belangrijk</p>
        - id attribute ID="id-waarde"
            <h1 id="myHeader">My Header</h1>
        - Title attribute TITLE="waarde”
            <p title="Free Web tutorials">W3Schools.com</p>
        - href attribute 
            <a href="https://www.w3schools.com">Visit W3Schools</a>

7. Wat is het verschil tussen inline en block elementen?

Een block-level element begint altijd op een nieuwe regel en neemt de volle breedte van het browservenster in. Het strekt zich uit naar links en rechts uit zo ver als het kan. Een inline element start niet op een nieuwe regel en beslaat slechts zo veel breedte als het nodig heeft om wat erin staat weer te geven. W3school Blocks

Inline: <span> <img> <a> The following span is an inline element; its background has been colored to display both the beginning and end of the inline element's influence.

Block: <div> <section> <p> The following paragraph is a

block-level element; en colored to display both the beginning and end of the block-level element's influence.

8. Wat betekent nesting? Codeer een voorbeeld met nesting erin.

Nesting betekent elementen die binnen elementen worden geplaatst.
It is a common practice to place multiple <div> inside another <div>. This is usually referred to as "nesting" elements and allows for further dividing elements into subsections or aid developers with CSS styling.

The <div class="outer-div"> is used to group together two <div class="inner-div"> elements; each containing a <p>element.
Bijvoorbeeld: 

        <div class="outer-div">
            <div class="inner-div"> 
                <p>This is a paragraph</p>
            <div> 
            <div class="inner-div">
                <p>This is another paragraph</p>
            </div>
        </div>
