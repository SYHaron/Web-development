Q14. Wanneer zou je een String met single quote ' gebruiken?

Je gebruik "" of '' als je text, getallen, speciale symbolen wilt gebruikten. Je gebruikt een combinatie als je verschillende dingen wilt aanduiden, bijvoorbeeld String las verkleining 'it is' naar "it's" Je gebruikt enkel 1x "" en/of 1x '' in String en die dubbel (2x "" of 2x'').
