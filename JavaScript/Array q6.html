q6.Zoek array functies, geeft voorbeeld en leg het uit.

push() array function. Adds new item toe achteraan de array. Vooraan gebruik unshift() methode/functie.
console.log => (5) ["aarbei", "peer", "framboos", "kokosnoot", "blauwe bes"] // blauwe bes is toegevoegd aan de array.

pop() verwijderd het laatste item in de array. Bij shift() is het eerste item in de array.
console.log => (3) ["aarbei", "peer", "framboos"] // kokosnoot is verwijderd.

shift() verwijderdt het eerste item in de array. De pop() verwijdert het laatste item in de array.
console.log => (3) ["peer", "framboos", "kokosnoot"] // aardbei is verwijdert.

slice() kan je items selecteren in de array. Maak er een var/let van.
console.log(new variable) => (2) ["aarbei", "peer"] //index 0 en index 2 gekozen.

splice() kan je items toevoegen & verwijderen aan een array.
console.log(splicefruit) => (3) ["citroen", "appel", "kokosnoot"] // aarbei/peer/fambroos verwijdert. citroen & appel toegevoed. index gekozen nummer is van 0 tot 3.

sort() arrays automatisch op alphabetische volgorde, maar kan ook bepaalde strings, up/down, nummers.
console.log(sortFruit) => (4) ["aarbei", "framboos", "kokosnoot", "peer"] // alphabetische volgorde.
