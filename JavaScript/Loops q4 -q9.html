q5. Maak dezelfde opdracht met 'for...in loop'. (array sports, 1 voor 1 in concole).

For/in loop loops through the properties of an object.
syntax: for (variable/property in object) {statement}. //bij voorkeur niet gebruiken bij arrays.
variable/property ==> different property name is assinged to variable on each iteration.
object => object whose non-symbol enumerable properties are iterated over.
console.log: 
                0: voetbal
                1: basketbal
                2: volleybal
                3: hockey
                4: honkbal
                5: zwemmen
                6: korfbal

q6. Maak dezelfde opdracht met 'for...of loop'. (array sports, 1 voor 1 in concole).

syntax: for (variable of iterable) {statement}.
variable => on each iteration a value of a different property is assigned to variable. Variable may be declared with const, let/var.
iterable =>object whose iterable properties are iterated.
